Axis System

Projet central de ma reconversion vers le développement web. Il présente mon parcours, mes projets et ma progression, tout en me servant de terrain d’apprentissage.

Présentation du projet

Axis System est mon portfolio professionnel, mais aussi mon projet d’apprentissage principal. Je le construis progressivement pour comprendre les bases du développement web : structure HTML, organisation des pages, CSS, responsive design, interactions JavaScript et déploiement.

Ce site n’est pas seulement une vitrine : c’est aussi mon terrain d’apprentissage.

Pourquoi “Axis System” ?

J’ai choisi ce nom pour représenter l’idée d’un axe central : un point qui regroupe mon parcours, mes projets, mes objectifs et ma progression. Le portfolio devient une interface centrale autour de ma reconversion professionnelle.

Objectifs du projet

  • Créer une présence professionnelle crédible.
  • Présenter ma reconversion vers le développement web.
  • Centraliser mes projets et ma progression.
  • Apprendre HTML, CSS et JavaScript sur un vrai projet.
  • Préparer un support clair pour mes candidatures.

Compétences travaillées

HTML

Structure de pages, balises sémantiques, navigation et organisation du contenu.

CSS

Mise en forme, composants, responsive, cartes, boutons et fichiers séparés.

JavaScript

Menu mobile, modale d’images, loader, animations et barres de progression.

Organisation

Dossiers, fichiers, GitHub, préparation au déploiement et documentation.

Méthode d’apprentissage

J’apprends en construisant. Sololearn me sert de support mobile pour revoir les bases et pratiquer régulièrement, tandis que VS Code et GitHub me permettent de tester sur un vrai projet.

Chaque erreur de chemin, chaque bug CSS ou chaque interaction JavaScript me donne une occasion de mieux comprendre la logique du web.

Déploiement prévu

Le site est pensé pour être publié proprement : GitHub pour le suivi du code, VPS pour l’hébergement, Nginx pour servir le site, puis domaine personnalisé et HTTPS.

À consolider

  • Préparer la mise en ligne sur le VPS.
