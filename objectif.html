Recherche alternance développement web

Construire mon avenir dans le développement web.

Mon objectif est d’intégrer une formation en développement web et de trouver une alternance afin d’apprendre en entreprise, progresser sur des projets concrets et construire progressivement un profil développeur web / full stack.

Mon projet professionnel

Après mon parcours en comptabilité, je souhaite me réorienter vers le développement web. Ce choix vient de mon intérêt pour la création d’interfaces, l’organisation de projets, la logique du code et la possibilité de transformer une idée en résultat concret.

Je veux apprendre sérieusement, avec méthode, en construisant des projets réels plutôt qu’en restant uniquement dans la théorie.

Ce que je recherche

  • Une alternance en développement web / full stack.
  • Une entreprise ouverte à un profil en reconversion.
  • Des missions concrètes et progressives.
  • Un cadre où apprendre les bonnes pratiques.
  • Une équipe capable de m’accompagner et de me faire progresser.

Pourquoi l’alternance ?

L’alternance représente pour moi le meilleur format pour apprendre le développement web : formation, pratique réelle, retours professionnels et projets concrets.

Je veux apprendre dans un environnement où le code sert à répondre à de vrais besoins.

Objectifs court terme

HTML

Structurer correctement les pages et comprendre les balises sémantiques.

CSS

Flexbox, Grid, responsive, composants et interface propre.

JavaScript

Interactions simples, DOM, modale, menu et progression.

GitHub

Projets publics, README, suivi du code et préparation du déploiement.

Objectifs moyen et long terme

  • Créer plusieurs projets présentables.
  • Apprendre React et mieux comprendre Node.js.
  • Découvrir les API et les bases de données.
  • Participer à un projet en équipe.
  • Évoluer vers un profil développeur web / full stack.

Ce que je peux apporter

Mon parcours en comptabilité m’a appris à être précis, à suivre des processus, à vérifier les informations et à travailler avec méthode. Ces qualités sont utiles pour relire du code, organiser un projet, corriger des erreurs et documenter proprement.
